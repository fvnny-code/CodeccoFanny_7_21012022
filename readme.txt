Pour lancer  l'application :

* Cloner le repository,


* Installer les back-end et front-end :
	npm install

* Créer la base de données à partir du fichier groupomania2.sql dans le dossier database côté back-end,

* Mettre à jour le fichier de connexion à la base de données si besoin (user, password, et port)


* Lancer le sever back-end : 
    nodemon server

* Lancer le server front-end 
    npm run serve


Dans le navigateur :
Lien pour afficher la page d'authentification :

 http://localhost:8080/login
